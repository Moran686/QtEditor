# QtEditor功能简述
#### 一个可以多窗口编辑的文本编辑器，支持了基本的记事本的作用

* 新建、保存
* 文本编写
* 复制粘贴
* 撤回重写

#### 也其中支持一些样式更改

* 字体颜色大小
* 窗口或编辑器背景色选择
* 对齐方式

#### 与记事本不同的是，我新增了查找和替换文本工具

* 在窗口中查找对应字段
* 对查找的固定字段进行文本替换
